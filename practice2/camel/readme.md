
# Camel to Snake Case Converter

Description - 

A Python program that converts variable names from camelCase to snake_case following Python's PEP 8 naming conventions.

Input: camelCase variable name (e.g., firstName, preferredFirstName)
Output: snake_case equivalent (e.g., first_name, preferred_first_name)

$ python camel.py
camelCase: firstName
snake_case: first_name

$ python camel.py
camelCase: preferredFirstName
snake_case: preferred_first_name
